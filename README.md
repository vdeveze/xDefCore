# xDefCore

Android app built with Kotlin + Gradle.

## Requirements

- JDK 11+
- Android Gradle Plugin 8.7.0
- Gradle 8.x

## Setup

```bash
./gradlew build
```

## Run

```bash
./gradlew installDebug
```

## Test

```bash
./gradlew test
```

## Project Structure

```
app/
  src/main/java/com/xdefcore/
  src/main/res/
build.gradle.kts
settings.gradle.kts
```
